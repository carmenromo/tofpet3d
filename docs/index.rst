tofpet3d: 3D Time-of-flight PET reconstruction
==============================================
This is a PET reconstruction code written in C++ with a Python
interface. It was originally developed within the framework of the project
TOF-Brain-PET and is used in the PETALO project (see `ANTEA <https://github.com/nextic/ANTEA>`_).
